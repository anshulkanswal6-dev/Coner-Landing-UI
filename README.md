# Coner AI - Growth Engine OS

A premium, high-converting landing page for Coner AI, built with React 19, Tailwind CSS 4, and Framer Motion.

## Features
- **Responsive Navbar**: Dark theme with glass effect and scroll states.
- **Hero Section**: High-impact headline with real-time CRM integration indicators.
- **Integration Section**: 60-second setup guide with a beautiful code terminal and universal compatibility.
- **Feature Performance**: Interactive WobbleCards highlighting core AI capabilities.
- **Accordion FAQs**: Clean interaction for technical and pricing questions.
- **Premium Footer**: Modern layout with social links and product navigation.

## Tech Stack
- **Framework**: React 19 (Vite)
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Setup Instructions

### 1. Integration
The product is live. Users can integrate Coner by adding the widget script to their website:

```html
<script
  src="https://cdn.coner.ai/widget.js"
  data-id="YOUR_UNIQUE_ID"
  async
></script>
```

### 2. Development
```bash
npm install
npm run dev
```

### 3. Deployment
```bash
npm run build
```
